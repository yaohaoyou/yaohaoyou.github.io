  1. rrr 数组无用
  2. 顺序正确的 lll 数组唯一
  3. 先从大到小排序，对于当前值 iii，放入第一个 lj=0l_j=0lj​=0 的位置，j>ij>ij>i。形成最后的 lll。

Code

递归求 fff 的主要代码如下：

void dfs(int l,int r){
    if(l>r)    return;
    int x=-query(all,l,r).se;  // 区间最小值的位置
    mmul(ans,C(r-l,x-l));
    dfs(l,x-1);
    dfs(x+1,r);
}
